# tournament
1. Download the project 
2. Open Postgres through terminal by :

				psql
3. Create new database with name "tournament"

				CREATE DATABASE tournament;
4. Connect to the database

				\c tourmament
5. Build up the database tables and views
			
			\i tourmament.sql
